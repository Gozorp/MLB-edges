About this dashboard

Static GitHub Pages dashboard for the mlb_edge project. Slate data is baked into docs/data/ via a GitHub Action on every push, so the dashboard is self-contained — no cross-origin fetches, works whether the source repo is public or private.

Grades are produced by an eight-rule grader: SP edge with small-sample dampening, PQI (bullpen quality), team-quality modifier, F-signal cap, compound-small-sample cap, Odds-API guard, PLATINUM gating, and Stage 1/2 agreement. See the README for the full rubric.

Autonomous updates: a daily workflow runs predict.py at 07:00 UTC, generates the new slate, and commits it. The dashboard auto-refreshes within a minute or two of the commit landing.

Interactive query: ask the slate about specific games or categories using the query box above. Searches the loaded CSV + parlay data — no LLM call, no network.
